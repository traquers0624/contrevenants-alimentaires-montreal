Documentation de l'API REST

Route: /contrevenants

Exemple: GET /contrevenants?du=2022-05-08&au=2024-05-15

Retourne une liste JSON des établissements ayant obtenu des contraventions entre deux dates, avec leur nombre respectif.

Route: /infractions

Exemple: GET /infractions?etablissement=ATLAS%20PIZZA

Retourne les détails des infractions pour un établissement spécifique.

Route: /doc

Affiche cette page de documentation avec le contenu du fichier RAML.

Spécification RAML

{{ raml }}

Route: /plainte

Exemple: POST /plainte

Permet de créer une plainte d’inspection. Le corps de la requête doit être un JSON valide (voir RAML ci-dessous).
